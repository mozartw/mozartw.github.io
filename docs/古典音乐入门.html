    • 西方音乐

        • 主要使用原声乐器
        • 以器乐为主；乐曲持续时间较长
        • 内涵厚重
        • 严肃音乐
      • 狭义

    • 巴洛克时期 （1600-1750）
      • 理论技法
        • 风格特征
          • 大量装饰性音符
          • 节奏强烈，旋律精致
          • 通奏低音，即兴创作
          • 复调音乐占主导地位（J.S.巴赫时达到极致），大小调取代教会调式；主调音乐蓬勃发展
          • 管弦乐团编制尚未标准化
        • 发展中心
          • 音乐创作的发展中心以贵族宫廷、私人组织的学会以及天主教教会为主，其中又以宫廷最具影响力
        • 音乐创作
          音乐创作进入蓬勃发展的阶段
          • 大协奏曲，协奏曲
          • 奏鸣曲，组曲，赋格曲
          • 歌剧——在威尼斯快速兴起
          • 弥撒，神剧（清唱剧；类似于歌剧的声乐，无布景、化妆、戏剧表演），受难曲
          • 管风琴曲目
          • 罗曼尼斯卡
      • 人物
        • 意大利
          • 阿尔坎杰罗·科雷利
            • 「现代小提琴技巧创建者」，「世界第一伟大小提琴手」，「大协奏曲之父」
            • 小提琴家，作曲家；几乎只为弦乐器创作
          • 朱塞佩·多梅尼科·斯卡拉蒂
            羽管键琴演奏家
          • 安东尼奥·卢奇奥·维瓦尔第
            • 外号“红发神父”；协奏曲之王
              出生当天遭遇地震
            • 作曲家，大师级小提琴演奏家；音乐教师（小提、大提、抒情维奥尔琴）
            • 主要成就是乐器协奏曲（尤其是小提琴协奏曲）
              • 1940年，维瓦尔第的作品被重新挖掘（法国音乐研究员马克·宾舍利等）
              • 协奏组曲《四季》
              • 将独奏协奏曲变成了巴洛克音乐的最佳表现形式；将三乐章作品带往成熟；
              • 维瓦尔第的慢乐章以悠扬如歌的旋律著称
          • 相关乐团
            • 欧洲嘉兰古乐团
            • 意大利协奏团
            • 和谐花园
            • 蒙多维皇家学院乐团
            • 马修斯合奏团
        • 德国
          • 格奥尔格（乔治）·弗里德里希·韩德尔
            • 作曲家，剧作家；出生于德国（神圣罗马帝国），成名且逝世于英国伦敦
            • 神剧（清唱剧）《弥赛亚》；管弦乐组曲《水上音乐》、《皇家烟火》
            • 擅长：歌剧，清唱剧，协奏曲
          • 约翰·塞巴斯蒂安·巴赫（Johann Sebastian Bach）
            • 作曲家；演奏家（管风琴、小提琴、大键琴）；
            • 巴洛克音乐的集大成者，将巴洛克音乐发展到巅峰
            • 对位法
            • 著名作品
              • 《G弦上的咏叹调》
              • 《平均律钢琴曲集》
              • 《勃兰登堡协奏曲》
              • 《哥德堡变奏曲》
              • 《马太受难曲》
              • 《赋格的艺术》
              • 无伴奏大提琴组曲；无伴奏小提琴奏鸣曲及组曲
            • 擅长
              • 管风琴曲、宗教音乐、协奏曲、独奏曲；多产作家
            • 对后世的影响
              • 西方“现代音乐”之父
              • 1829年，门德尔松指挥上演《马太受难曲》，宣告“复兴巴赫”运动的开始。
              • 浪漫主义时期的作曲家多推崇巴赫
              • 20世纪，由于新古典主义音乐的兴起，巴赫风格在现代音乐创作中的地位大大提升
    • 古典主义时期 （1750-1820）
      • 风格特征
        • 出现了多乐章交响曲、独奏协奏曲、弦乐四重奏、多乐章奏鸣曲等体裁
        • 奏鸣曲式和回旋曲式成为最常见的曲式（古典和浪漫时期）
        • 乐团编制增大，乐团由指挥带领成为常规
        • 现代钢琴出现，逐渐取代大键琴的地位
        • 作曲家逐渐转变为独立经营
      • 人物作品
        • 早期
          • 弗朗茨·约瑟夫·海顿（奥地利）
            • 作曲家；绰号“海顿爸爸”；
            • 性格相貌
              • 幽默，爱开玩笑；乐观，善解人意
              • 虔诚的天主教徒
              • 个子不高，面貌不佳
              • 爱好打猎、钓鱼
            • 主要成就
              • 巴赫之后第一位伟大的器乐作曲家；
              • 被尊称：交响乐之父，弦乐四重奏之父
              • 将奏鸣曲式从钢琴发展到弦乐重奏上；发明了奏鸣回旋曲式
              • 器乐主调的创始人；钢琴协奏曲和钢琴三重奏的开创者
              • 德国国歌《德意志之歌》的作曲者
            • 音乐风格
              • 幽默，明快，有一种宗教式的超脱
            • 主要作品
              • 清唱剧《创世记》
              • 大量交响曲和室内乐作品
            • 社会关系
              • 受人尊敬、追捧
              • 与忘年交、好朋友沃尔夫冈·阿马德乌斯·莫扎特感情很好
              • 学生路德维希·凡·贝多芬
        • 中期
          • 沃尔夫冈·阿马德乌斯·莫扎特（奥地利/神圣罗马帝国）
            • 天才音乐家，自幼成名
        • 晚期
          • 贝多芬
          • 舒伯特
          • 帕格尼尼
