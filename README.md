# 以下是支持的语法  
***  
***  
## 文字和段落：
1. 文字加粗：  
   输入代码：`**加粗**`   
  输出效果：**加粗**  
2. 文字倾斜：  
  输入代码：`*倾斜*`  
  输出效果：*倾斜*
3. 文字同时加粗和倾斜：  
  输入代码：`***加粗并且倾斜***`  
  输出效果：***加粗并且倾斜***
4. 标题：  
  输入代码：`# 一级标题`  
  输出效果：  
  # 一级标题  
  输入代码：`## 二级标题`  
  输出效果：    
  ## 二级标题  
  输入代码：`###### 六级标题`  
  输出效果：    
  ###### 六级标题    
5. 引用：  
  输入代码：  `> 伟大领袖毛主席曾经说过：“没有调查就没有发言权！”`  
  输出效果：  
  > 伟大领袖毛主席曾经说过：“没有调查就没有发言权！”  
6. 链接：  
  输入代码：  
  `[百度一下](https://www.baidu.com/)`  
  输出效果：  
  [百度一下](https://www.baidu.com/)  
  图片链接：  
  输入代码：  
  `![百度LOGO]()`  
  输出效果：  
  ![百度LOGO]()  
  ***  
7. 列表：    
输入代码：  
```  
* 无序列表 1
* 无序列表 2
* 无序列表 3
1. 有序列表 1
2. 有序列表 2
3. 有序列表 3  
```  
---  
输入效果：  
* 无序列表 1
* 无序列表 2
* 无序列表 3
1. 有序列表 1
2. 有序列表 2
3. 有序列表 3
